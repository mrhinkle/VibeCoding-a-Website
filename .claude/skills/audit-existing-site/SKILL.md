# Audit Existing Site

You already have a website—WordPress, Squarespace, Wix, custom build, doesn't matter. Before you think about rebuilding, let's see what you've actually got.

**This skill analyzes your live website** without requiring any repo access, GitHub connection, or code handoff. We fetch your site and run a comprehensive 4-part audit: SEO & AEO, Copy & Content, UX, and Security. You get a health score, quick wins, and a roadmap for improvement.

Think of this as a **gateway skill**. See the value? Then explore how VibeCoding could rebuild or enhance your site from the ground up.

## What You Need

- **A live URL** (your existing website, already published on the internet)
- That's it. No repo access, no GitHub, no code to share.

## The Audit Pipeline

We run 4 specialized audits in sequence, each targeting a different dimension of site health:

### 1. SEO & AEO Audit
Searches for technical and structural SEO signals:

- **Meta tags & title tags**: Are they present, unique, and keyword-rich?
- **Open Graph / social sharing tags**: Will your site look good when shared?
- **Heading hierarchy**: H1 → H2 → H3 structure (no orphans, no skips)
- **Schema markup & JSON-LD**: Rich snippets for Google
- **Mobile-friendliness signals**: Viewport tags, responsive patterns
- **Page speed indicators**: Core Web Vitals, load time estimates
- **Alt text on images**: Every image described for accessibility and SEO
- **URL structure**: Readable slugs, no parameters, proper trailing slashes

### 2. Copy & Content Audit
Reviews your actual words and messaging:

- **Headline effectiveness**: Do they hook? Is the benefit clear?
- **CTA strength and placement**: Are calls-to-action specific, urgent, and visible?
- **Readability score**: Target is 6th–8th grade level. Shorter sentences win.
- **Value proposition clarity**: Could a stranger understand your value in 5 seconds?
- **Microcopy review**: Button labels, form fields, error messages, help text
- **Tone consistency**: Is your voice uniform across pages?

### 3. UX Review
Examines user behavior and interface design:

- **Navigation depth**: Can users find what they need in 3 clicks or fewer?
- **Mobile responsiveness**: Does your site work on phones, tablets, desktops?
- **Visual hierarchy**: Is the most important content the most prominent?
- **Touch target sizes**: All buttons and links ≥44×44px (mobile standard)
- **Content readability**: Line length (50–75 characters ideal), font size (16px+)
- **Common anti-patterns**: Autoplaying video, pop-ups on load, unclear submit buttons, etc.

### 4. Security Surface Scan
Checks publicly visible security indicators:

- **HTTPS enforcement**: Are all pages served over SSL/TLS?
- **Security headers**: CSP, HSTS, X-Frame-Options, X-Content-Type-Options
- **Mixed content warnings**: No HTTP assets loaded on HTTPS pages
- **Third-party script inventory**: What external scripts are you running?
- **Cookie & privacy policy presence**: Are legal requirements met?

## What This Audit CAN'T Check

Be transparent about limitations. A remote URL audit doesn't see:

- **Server-side code quality**: Node, Python, PHP, database queries—we don't audit those
- **Database security**: Schema, SQL injection risks, data encryption—out of scope
- **Backend dependencies**: NPM packages, plugins, libraries—we don't inspect these
- **Full JavaScript execution**: We analyze the HTML/CSS and limited JS, not dynamic state
- **Authenticated pages**: We can't log in, so admin panels, member dashboards are invisible
- **Internal link structure**: We sample public pages, not your entire sitemap
- **Performance under load**: We test from one location, not across global networks

If you need those deeper audits, this is where VibeCoding shines—we can rebuild from the ground up with security and scalability baked in.
