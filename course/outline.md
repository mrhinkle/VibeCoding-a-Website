# VibeCoding: Building Websites with Claude Code + Manus
## Complete Course Outline

Welcome! This course teaches you how to build professional websites without hiring an agency, without struggling with code, and without template limitations. You'll learn to "vibe code"—working with AI and design tools to create exactly what you envision.

---

## Course Structure

**Core Track (1 hour)** – The essentials to ship a live website fast.
**Expanded Track (3–4 hours)** – Deeper dives into customization, design, and ongoing maintenance.

Each module lists its core time + expanded time. You can mix and match: do core for everything, or expand on topics you care most about.

---

## Module 00: Introduction (5 min)
**What you'll learn:**
- What "vibe coding" means and why it matters for non-developers
- How the old way (agencies, templates, DIY code) compares to the new way (AI-assisted)
- What you'll build by the end of this course (your live website)
- What you actually need to get started
- The two tools you'll use and why they matter

**Key takeaway:** You have the right tools now. Building a website is no longer the barrier it used to be.

## Module 01: Setting Up Your Tools (5 min core | 15 min expanded)
**Core (5 min):**
- Installing Claude Code
- Getting an Anthropic API key
- Creating a GitHub account

**Expanded (10 min more):**
- Installing Git and GitHub Desktop (non-terminal friendly)

- Running a test command to verify everything works

**Key takeaway:** Setup takes less time than you'd expect. We've made this simple.

## Module 02: The Discovery Interview (8 min core | 23 min expanded)
**Core (8 min):**
- Why discovery matters (quality input = quality output)
- Running the site-interviewer skill

- Understanding your site-brief.json output

**Expanded (15 min more):**

## Module 03: Brand & Visual Assets (10 min core | 35 min expanded)

- Understanding the image prompts Claude generates

**Expanded (25 min more):**
- Creating brand guidelines from scratch (no designer needed)

- Typography basics and choosing fonts

**Key takeaway:** Professional-looking images are easier to create now. You don't need a designer—you need the right tool.

## Module 04: Generate Your Website (12 min core | 42 min expanded)
**Core (12 min):**
- Running the site-generator skill

**Key takeaway:** Your website code is readable and customizable. It's not a black box.

## Tips for Success

**You don't need:**
- Coding experience
- Design experience
- Thousands of dollars
- A team

**You do need:**
- A computer with internet
- One hour to two hours of focused time
- An idea (even a rough one)
- Curiosity

**The mindset that helps:**
- You can iterate. First draft won't be perfect, and that's fine.
- AI tools are fast. Use them as many times as you need.
- Breaking things is okay. We'll show you how to fix them.
- This is a skill you're building. It gets easier.
