# Feedback & Issues

VibeCoding a Website is maintained by [Mark Hinkle](https://github.com/mrhinkle) and [The AI Enterprise](https://theaienterprise.io/). This project is provided as a free, open-source resource for the community.

## We're Not Accepting Pull Requests

This isn't a community-contribution project. We maintain the skills, course content, and examples ourselves to keep quality consistent. You're welcome to fork the repo and build on it for your own purposes under the MIT license.

## Reporting Bugs & Suggesting Improvements

If something is broken or you have an idea that would make VibeCoding better, open a ticket:

**[Open a GitHub Issue](https://github.com/mrhinkle/VibeCoding-a-Website/issues)**

When filing a ticket, include:

- **What you were doing** — Which skill, what step
- **What went wrong** — Error message, unexpected behavior, or what you expected instead
- **Your setup** — OS, Claude Code version, browser (if relevant)
- **Screenshots or logs** — If you have them

We'll review tickets and address them as time allows.

## Building Your Own Skills

Want to extend VibeCoding with your own skills? The [Skill Development Guide](./skill-development-guide.md) explains the SKILL.md format, directory structure, and how skills chain together. Fork the repo, add your skills, and use them locally with Claude Code.
